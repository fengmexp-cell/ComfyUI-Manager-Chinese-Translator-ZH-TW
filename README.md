# ComfyUI-Manager 東北亞多語直覺引擎 Pro Max v4.0

本倉庫提供 ComfyUI-Manager 的 **東北亞四語全自動直覺翻譯引擎**。專為繁體中文、簡體中文、日本語、한국어 使用者打造，專治各種原廠介面不服與疑難雜症，大幅提升 UI 實戰體驗！

## ✨ 核心特色 (Features)
- 🌐 **四語大一統**：一個腳本，自動識別瀏覽器語系切換 **繁中 / 簡中 / 日文 / 韓文**，免去手動換檔的麻煩。
- ⚡ **直覺翻譯引擎**：拒絕生硬的機器翻譯，完全採用最接地氣、最直覺的實戰職人用語。
- 🚨 **動態資安攔截**：完美降伏遠端動態載入的英文公告（如惡意套件警告、版本更新資訊），不留多餘的英文尾巴，保持介面絕對純淨。

---

## 💾 安裝說明 (Installation)

### 1. 下載腳本 (Download File)
請下載本倉庫的終極大一統腳本檔案：
👉 **`comfyui_manager_northeast_asia_translator.js`**

### 2. 放置路徑 (Place Into)
將下載的 `.js` 檔案直接放入你的 ComfyUI 資料夾中：
`ComfyUI/custom_nodes/ComfyUI-Manager/js/`

### 3. 重整瀏覽器 (Refresh Browser)
打開 ComfyUI 網頁，按下鍵盤上的強制重整快捷鍵：
👉 **`Ctrl + F5`** (Mac 使用者請按 `Cmd + Shift + R`)

### 4. 安裝完成 (After Installation)
重新點開 ComfyUI-Manager，系統將會根據你的瀏覽器語言，自動套用最完美的在地化直覺介面！
